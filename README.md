# TMSimple

Simple Test Case Management System

Self-Study project to update knowledge about:

-   Nodejs
-   Express
-   Mongodb
-   Jest
-   ESLint
-   Prettier
-   Docker
-   Github Workflows

## Getting started

1. Start the `tmsimple server` service (and any others) in the background:

    ```bash
    docker-compose up -d
    ```

1. The server is running on <http://localhost:3000>.
